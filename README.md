# orcrist-ui

OrcristJS Family : React Web Components.

## 代码风格

ReactJS & TypeScript

```bash
$ npm run lint
```

## 开发流程

```bash
$ npm install
$ npm start
```

### 提交代码

```bash
$ npm publish
```

### 使用组件

```js
import { YourComponent } from 'orcrist-ui';
```
